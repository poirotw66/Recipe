# Recipe audit reports

Human-in-the-loop QA for `src/content/recipes/`. Generated by the **recipe-review** Cursor skill (`.cursor/skills/recipe-review/`).

## Files

| Path | Purpose |
| --- | --- |
| `latest-precheck.json` | Mechanical checks from `npm run recipe-review:precheck` |
| `reports/{slug}.json` | Full per-recipe review (agent + precheck merged) |
| `audit-summary-YYYY-MM-DD.md` | 批次摘要（**繁體中文**） |
| `audit-summary-*-sample-10.md` | 10 篇範例審查摘要 |
| `overrides.json` | Human-approved false positives (optional) |

## Commands

```bash
npm run recipe-review:full          # 100 篇：規則審查 + reports + 繁中 summary
npm run recipe-review:precheck      # 僅機械 precheck
node scripts/recipe-review-precheck.mjs --slug tomato-egg-rice --write
```

## Cursor

聊天輸入 **`/recipe-review`** 會透過 `.cursor/hooks/recipe-review.sh` 觸發全站審查（需啟用專案 Hooks）。

## Status meanings

- **Pass** — No actionable issues
- **Warning** — Fix when convenient; agent may suggest text edits
- **Critical** — Safety or major logic; human must confirm before ignore
